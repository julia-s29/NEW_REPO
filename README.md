# Practice Tasks 

Проект с практическими заданиями по API и UI тестированию на Python. 

## Технологии 
- Python 
- Pytest 
- Requests (API тесты) 
- Playwright (UI тесты) 


## Структура проекта 
tests/api/ 
- test_delete.py 
- test_get.py 
- test_post.py
- test_put.py 

tests/ui/ 
- test_login.py 

conftest.py 

requirements.txt 

README.md 


## Установка 
pip install -r requirements.txt 

playwright install 


## Запуск тестов 
pytest -v
